ChatPilot - Documentation

Thank you for buying ChatPilot. This guide will help you install and configure your new AI-powered WhatsApp SaaS platform.

1. Server Requirements

Ensure your server meets the following requirements:

  • PHP >= 8.1
  • MySQL >= 5.7 or MariaDB
  • BCMath PHP Extension
  • Ctype PHP Extension
  • JSON PHP Extension
  • Mbstring PHP Extension
  • OpenSSL PHP Extension
  • PDO PHP Extension
  • Tokenizer PHP Extension
  • XML PHP Extension

2. Installation Steps

  1. Upload Files: Upload the contents of the Main_File folder to your server's public_html or root directory.
  2. Install Dependencies: Run the following command in your terminal: 
    composer install --optimize-autoloader --no-dev
  3. Environment Setup:
    • Rename .env.example to .env.
    • Open .env and update your database credentials: 
      DB_DATABASE=your_database_name
DB_USERNAME=your_database_user
DB_PASSWORD=your_database_password
  4. Generate Key: Run: 
    php artisan key:generate
  5. Run Migrations: Setup your database tables: 
    php artisan migrate --seed
  6. Link Storage: 
    php artisan storage:link

3. Supervisor / Queue Setup (Critical)

This step is required for Campaigns to send messages in the background.

You need to ensure php artisan queue:work is running constantly. On Linux servers, install Supervisor:

[program:chatpilot-worker]
process_name=%(program_name)s_%(process_num)02d
command=php /path/to/your/project/artisan queue:work --sleep=3 --tries=3
autostart=true
autorestart=true
user=root
numprocs=1
redirect_stderr=true
stdout_logfile=/path/to/your/project/storage/logs/worker.log

4. Facebook App Credentials

To connect to WhatsApp, you need a Meta App:

  1. Go to Meta for Developers.
  2. Create a "Business" app.
  3. Add the "WhatsApp" product.
  4. Get your Permanent Access Token and Phone Number ID.
  5. Enter these details in the ChatPilot Connect Device page.
